# Shooter
A space arcade shooter developed in Python using the Pygame library.
The player controls a spaceship that can move left and right across the screen. The goal is to score 30 points by shooting down UFOs. Colliding with an enemy or asteroid results in a loss of one life. Running out of lives (3 lives) or missing 5 enemies results in defeat.
The project contains the following files:
1. shooter_game.py    main game file
2. shooter_game.exe   game file
3. galaxy.jpg         background image of space
4. rocket.png         image of the player (rocket)
5. ufo.png            image of the enemy (UFO)
6. bullet.png         image of bullet
7. space.ogg          background music
8. fire.ogg           sound of a gunshot
